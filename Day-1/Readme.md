What is print()?

The print() function in Python is used to display output to the console.
It is one of the most basic and widely used functions for debugging, logging, and displaying information.

examples:
print("helloworld")
print("am a software engineer")
print("Starting system check...")
print("CPU Status: OK")
print("Memory Status: OK")
print("Disk Status: OK")
print("System Healthy")
